# Secret Garden 🌸

> 우리의 하루가 꽃이 됩니다.

오늘의 마음을 한 송이 꽃으로 남기는 비밀의 정원.
Next.js 14 (App Router) + Tailwind CSS로 만들어졌습니다.

---

## 🌱 로컬에서 실행하기

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000` 으로 접속합니다.

---

## 🚀 Vercel 배포하기

이 저장소는 별도 설정 없이 Vercel에 바로 배포할 수 있습니다.

1. **GitHub에 푸시** — 이 폴더를 GitHub의 새 저장소에 업로드합니다.
2. **Vercel 접속** — [vercel.com](https://vercel.com) 에 로그인 (GitHub 계정으로 가능)
3. **새 프로젝트 생성** — Add New… → Project → GitHub 저장소를 Import
4. **Deploy 버튼 클릭** — 환경변수나 추가 설정 없이 바로 배포됩니다.

배포가 끝나면 `your-project.vercel.app` 형태의 주소가 발급됩니다.

---

## 💾 데이터 저장 방식

기본 설정에서는 브라우저의 **localStorage**에 꽃을 저장합니다.

- ✅ 같은 브라우저에서는 정원 코드를 입력하면 다시 볼 수 있습니다.
- ✅ 같은 브라우저의 다른 탭에서는 실시간으로 공유됩니다.
- ⚠️ 서로 다른 기기·브라우저 사이에서는 자동으로 공유되지 않습니다.

여러 기기에서 같은 정원을 공유하려면 Vercel KV를 붙이세요 (아래 참고).

---

## 🔄 (선택) Vercel KV로 진짜 공유 정원 만들기

1. `npm install @vercel/kv`
2. Vercel 대시보드 → 프로젝트 → **Storage** → **Create Database** → **KV**
3. KV를 프로젝트에 **Connect** (환경변수 자동 주입)
4. `app/api/garden/[room]/route.js` 에 GET/POST 두 가지 API 작성
5. `app/page.jsx` 의 `localStorage` 호출을 `fetch('/api/garden/...')` 로 교체

이 작업이 부담된다면, "Vercel KV로 바꿔줘"라고 다시 요청해 주세요.

## 📂 폴더 구조

```
secret-garden/
├── app/
│   ├── components/
│   │   ├── constants.js       # 8 감정 + 160종 꽃 풀 + 조사 헬퍼
│   │   ├── Flowers.jsx        # 5종 SVG 꽃, 이모지 꽃, 꽃망울, RenderFlower
│   │   ├── Fireflies.jsx      # 떠다니는 반딧불
│   │   ├── WelcomeScreen.jsx  # 이름·코드 입력 화면
│   │   └── GardenScreen.jsx   # 정원 + 꽃 피우기 + reveal 모달
│   ├── globals.css            # Tailwind + 폰트 + 애니메이션
│   ├── layout.jsx             # 루트 레이아웃
│   └── page.jsx               # 메인 페이지 (상태 관리)
├── next.config.mjs
├── tailwind.config.js
├── postcss.config.js
├── package.json
└── .gitignore
```
